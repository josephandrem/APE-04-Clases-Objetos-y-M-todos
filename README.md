# Sistema de Control de Estudiantes — C++ y Java

Sistema desarrollado como práctica de Programación Orientada a Objetos para la asignatura Algoritmos y Lógica de Programación.
El proyecto implementa un sistema básico de registro y control de estudiantes utilizando clases, objetos y métodos en los lenguajes C++ y Java.

## Características
 - Registro de estudiantes.
 - Ingreso de:
   - Cédula
   - Nombre
   - Apellido
   - Nota 1
   - Nota 2
   - Nota 3
 - Validación de notas entre 0 y 10.
 - Cálculo automático del promedio.
 - Determinación del estado:
   - Aprobado
   - Reprobado
 - Visualización del listado completo de estudiantes.
 - Resumen final con cantidad de aprobados y reprobados.


## Tecnologías utilizadas
 - C++
 - Java
 - Programación Orientada a Objetos (POO)
 - Git y GitHub
 - VS Code / Code::Blocks
